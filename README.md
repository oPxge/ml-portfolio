
# ML Portfolio — House Prices (Regression) & Company Bankruptcy (Classification)

## Structure
- `data/` — raw CSVs (`regression.csv` = kc_house_data, `classification.csv` = CompanyBankruptcyData)
- `notebooks/` — Jupyter notebooks:
    - `01_regression_house_prices.ipynb`
    - `02_classification_bankruptcy.ipynb`
- `reports/` — auto-saved artifacts + `final-report.md`
- `src/` — (optional) helper modules
- `requirements.txt`, `.gitignore`

## Quickstart
```bash
pip install -r requirements.txt
jupyter notebook
```
Open each notebook and run all cells. Outputs will be written to `reports/`.
